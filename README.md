# social

_Social media for the **user** <u>not</u> the company_

# How to use:

## Install dependiencies

### For server

```
cd server
npm i
```

### For client

```
cd client
npm i
```

## Config the .env

```
cd server
```

Then rename the **.env_example** to **.env**

Configure it to fit your setup

## How to run the server

```
cd server
npm run dev
```
